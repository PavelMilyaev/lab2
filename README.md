# Калькулятор расхода ткани и стоимости пошива

## Описание
Приложение для расчёта расхода ткани, стоимости пошива и материалов для различных видов одежды:
- Пиджаки
- Брюки
- Костюмы-тройки

## Возможности
- Добавление различных предметов одежды с параметрами
- Расчёт расхода ткани в зависимости от размера и особенностей
- Расчёт стоимости пошива
- Расчёт стоимости материалов
- Создание костюмов из отдельных предметов
- Сохранение результатов в базу данных SQLite
- Экспорт отчётов в формате .docx
- Просмотр истории расчётов
